# Project Summary

## Paper

Anatomy: Simple and Effective Privacy Preservation

## Objective

This project studies the paper "Anatomy: Simple and Effective Privacy Preservation".

The objective is to understand and implement the Anatomy privacy-preserving technique for publishing sensitive datasets.

Unlike traditional generalization methods, Anatomy separates quasi-identifiers and sensitive attributes into two independent tables:

* Quasi Identifier Table (QIT)
* Sensitive Table (ST)

The project will implement a simplified version of the Anatomy algorithm using Python and compare its performance against traditional generalization approaches.

## Evaluation Metrics

* Privacy preservation
* Information loss
* Query accuracy
* Runtime performance
